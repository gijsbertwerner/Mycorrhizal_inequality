Mycorrhizal Inequality Paper - Data Analysis
Gijsbert Werner, University of Oxford
July 2018

This folder contains all the data and scrits to reproduce the full analyses from Whiteside et al.

All data files are available as csv-files in the 'Data' folder. 

All analyses were peformed in R 3.4.4, using R-packages ggplot2 (version 3.0.0), reshape2 (1.4.3), nlme (3.1-137), car (3.0.0), ggpubr (0.1.7), gridExtra (2.3) and dplyr (0.7.6). 

Instructions:
- Set the unzipped folder Data_Scripts_Submission as your working directory in R
- Execute the R-script Analyses_Figures.R to generate two folders (Figures and Models), containing all paper figures and evaluated models.
- Execute the markdown file Analyes_Figures_Report.rmd to generate the full markdown-report detailing all these analsyes.
