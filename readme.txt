Mycorrhizal Inequality Paper - Data Analysis
Gijsbert Werner, University of Oxford
November 2018

This folder contains all the data and scripts to reproduce the full analyses from Whiteside et al.

All data files are available as csv-files in the 'Data' folder (data folder to be uploaeded to GitHub upon acceptance). 

All analyses were peformed in R 3.4.4, using R-packages ggplot2 (version 3.1.0), reshape2 (1.4.3), nlme (3.1-137), car (3.0.2), ggpubr (0.2), gridExtra (2.3) and dplyr (0.7.8). 

Instructions:
- Set the unzipped folder Data_Scripts_Submission as your working directory in R
- Execute the R-script Analyses_Figures_Nov2018.R to generate two folders (Figures and Models), containing all paper figures and evaluated models.
- Execute the markdown file Analyes_Figures_Report_V2.rmd to generate the full markdown-report detailing all these analsyes.
